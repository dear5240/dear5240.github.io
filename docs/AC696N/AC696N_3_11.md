# 11. 调音指南

> 目前客户对音箱产品品质要求越来越高，产品品质提升最明显的地方就是音箱的音质。本文档针对 69 系列音质调试的方法做一个介绍，提高音质调试的效率。

## 11.1. 调试步骤

> 目前调试最多的就是：客户以某音箱为参考样机，要求调试出相同的音质效果。

以下调试步骤以有参考样机为例子，对调试方法进行介绍。

![img219](/pic/219.png)



**① 硬件电路是否一致**

确认待调样机与对比样机的硬件电路是否完全相同。调音电路、功放电路参数等。

**注意 1：**功放型号、功放的 AB/D 类，是否有升压，防破音模式是否一样；

**注意 2：**DAC 输出到功放输入之间的电容大小是否一样；

![img220](/pic/220.png)



**② 喇叭、腔体、振膜、电池是否一致**

确认待调试样机的腔体跟喇叭是否完全一样，最好共用同一套腔体，用开关进行切换；确认电池电压是否一样，功放供电电压是否一样。



**③ DAC 输出最大幅度是否一致**

测试对比样机与待调样机 DAC 输出最大幅值是否一样。

测试方法 1：用音频分析仪测试 DAC 输出最大幅度。

测试方法 2：蓝牙播 1KHz@0dB 正弦波信号，示波器测试输出峰峰值。将待调样机输出峰峰值调得跟对比样机最大输出幅值一样。



**④ 对比频响曲线（20Hz-20KHz）**

测试方法：用音频分析仪直接扫描出频响曲线，对比两个样机的频响曲线，用在线调EQ 的方式，把待调样机的频响曲线拉得接近对比样机。

注意：有些样机开了 DRC 功能，最大音量的频响曲线是一根直线，看起来像是没有调过EQ。需要把音量减小一半，才能扫描出真正的 EQ 曲线；



**⑤ 结合实际声音效果，进行微调**

对比参考样机，听出两者之间的差异，进行微调。

好的音质效果具备：低频饱满，人声清晰，高频响亮。



## 11.2. 调音问题

（不同的喇叭腔体表现出来的效果不一样，经验之谈，仅做参考）

**① 声音没有低频效果，声音很干？**

答：提高低频，60-130Hz 频段，低频有深度；



**② 声音有点发散，不够集中？**

答：提高低频 250Hz 左右频段，提高 800-1K 频段；



**③ 声音有点闷、人声不够清晰？**

答：低频过多，中频不够。把 250HZ 附近的增益压一下，把 1KHz、3KHz、6KHz 附近的增益抬高，Q 值加大，10K 以后的增益不要压制太多。



**④ 播放某些歌曲，声音听起来很吵？**

答：高频过多，把高频的增益压一下，低频也可以尝试轻微压一下。



**⑤ 声音没有层次感，人声和伴奏背景音都混在一起？**

答：提高低频 70-100Hz 频段，衰减 200-400Hz 频段，提高 800-1K 频段，8K 以后的频段尽量不要压制太多；



**⑥ 播放某些低频、高频歌曲，出现杂音、喇叭打边的振动？**

答：低频或高频破音，把相应频点的增益降低。



**⑦ 类似鼓声、镲等乐器声音延续效果时间比较短？**

答：提高低频 70-100Hz 频段。

50HZ-100HZ：增加鼓声等低频音效的饱满度；

200HZ-400HZ：增加人声的饱满度，减少人声和中音乐器的浑浊不清；3KHZ：增加人声的清晰度、钢琴音的力度；

7KHZ：增加打击乐器的声音力度；

10KHZ 以后：增加人声、乐器的明亮度，过大声音会显得很吵。
